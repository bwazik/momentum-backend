June 2026 — Pre-Implementation Visual Concept

Screen Concepts

16 core screens designed to visualize the product before implementation. Click any screen to open the full mockup.

Executive Dashboard
Screen 01 Minister / Undersecretary

Executive Dashboard

High-level strategic overview with RAG status indicators, department SLA health bars, bottleneck analysis, escalation feed, and task aging distribution. The single pane of glass for top leadership.
Task Board
Screen 02 All Users

Task Board

Unified, filterable table view of all active tasks. Quick filters by status, department, priority, and blueprint. Sortable columns with SLA health indicators and inline assignee avatars.
Task Details
Screen 03 Task Participants

Task Details

Deep dive into a single task. Vertical stage timeline with SLA status, stage actions (submit/advance/return), threaded comments, document attachments, activity log, and external reference links.
Blueprint Builder
Screen 04 Admin / Process Owner

Blueprint Builder

Visual workflow editor with connected stage nodes. Left canvas shows the flow diagram; right properties panel allows configuring stage name, type, SLA, assignment method, exit requirements, and return/advance transitions.
Workflow Visualization
Screen 05 All Users

Workflow Visualization

Horizontal flow diagram showing a task's full lifecycle. Color-coded stage cards (completed/active/pending/overdue) with assignees, SLAs, sub-stage details, return path indicators, and a timeline bar.
Follow-Up Center
Screen 06 Follow-Up Specialist

Follow-Up Center

Dedicated monitoring hub for follow-up specialists. Scoped task board, live alert banner, bottleneck analysis by department, and manual follow-up action logging to replace spreadsheets and WhatsApp coordination.
Organization Structure
Screen 07 Admin / HR

Organization Structure

Interactive org chart tree from Minister down to section level. Visual hierarchy with gradient avatar nodes, department cards showing position counts and occupancy, and drill-down into sections.
Analytics & Reporting
Screen 08 Executives / Directors

Analytics & Reporting

Data-rich analytics hub with multiple report tabs (Overview, Department Performance, Blueprint Analysis, SLA Compliance, Task Aging, Workload). Bar charts, donut charts, trend graphs, and compliance tables.
System Administration
Screen 09 System Admin

System Administration

Central configuration panel with sidebar navigation (Organization, Users, Permissions, Calendar, Blueprints, Audit Log, Branding, Settings). Working calendar config, public holidays, and security classification management.
Help Center
Screen 10 All Users

Help Center

Built-in knowledge base with hero search, category cards (Getting Started, Managing Tasks, Blueprint Guide, Reports & Analytics), featured articles, and a browsable article library with metadata.
Manager Dashboard
Screen 11 Department Manager

Department Manager Dashboard

Team-focused operational dashboard for department heads. Shows workload snapshot, pending approvals, team activity feed, availability, and SLA health — all scoped to the manager's team.
Pending Approvals
Screen 12 Managers / Directors

Pending Approvals

Unified hub of all tasks awaiting manager decision. Filterable table with approve/return actions, SLA context, task details, and submission history. Includes approval statistics and recent decisions log.
Team Workload
Screen 13 Department Manager

Team Capacity & Workload

Visual workload distribution across team members with capacity indicators, overload warnings, availability tracking, and suggested task reassignments for balanced workload management.
Staff Performance
Screen 14 Department Manager

Staff Performance Hub

Individual performance scorecards with SLA compliance, completion rates, average resolution time, and overdue counts. Includes team trend charts and department-wide KPI summaries.
Decisions Registry
Screen 15 All Managers

Decisions Registry

Searchable record of management decisions, meeting outcomes, escalations, and directives. Each entry is linked to relevant tasks with decision maker, description, and pending action items.
Department Memos
Screen 16 Department Managers

Department Memos & Announcements

Internal communications hub for department-level memos, circulars, announcements, and meeting agendas. Features read tracking, priority indicators, audience targeting, and compose functionality.

Design System Summary

Technology

  • Next.js (React)
  • TypeScript
  • Tailwind CSS
  • shadcn/ui

Typography

  • Font: Inter
  • Headings: 600–700 weight
  • Body: 400 weight
  • Labels: 10–12px uppercase

Color Palette

Slate 900 — Sidebar
Emerald — Primary
Amber — Warning
Red — Danger

Inspiration

  • Linear (clean UI)
  • Jira (enterprise depth)
  • Monday.com (boards)
  • Notion (docs)
